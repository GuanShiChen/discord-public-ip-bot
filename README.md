# IP Monitor Discord Bot

This is a simple Discord bot that monitors your public IP address and sends a message to a specified channel when it changes. It's designed to be used with a VPN or proxy server where the public IP address may change frequently.

## Features

- Automatically detects your public IP address and sends a message to a specified channel when it changes.

## Requirements

- Python 3.11 or higher
- A Discord bot token
- A Discord channel ID
